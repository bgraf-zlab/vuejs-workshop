# workshop

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### With Docker:

i.e.
```
docker run -it -v $(pwd):/app -w /app -p 8080:8080 node npm run serve
```
